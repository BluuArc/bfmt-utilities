Basic Browser Example

Instructions

This is a basic example of how to use bfmt-utilities within the browser directly. The key thing you will want to do is import the script like so:

<script src="https://joshuacastor.me/bfmt-utilities/dist/index.browser.min.js"></script>

(Alternatively, you could use index.browser.js if you'd like to view an unminified version.)

After adding the script, you will now have access to the bfmtUtilities variable within your JavaScript, which is the entry point for all the functions.

The demo below fetches the latest copy of the unit datamine for Global from Deathmax's repository and displays the effects of each unit separated by source.

The source code can be found here.

  • The app.js file contains the logic for hooking up the button, checkbox, and dropdown as well as fetching the data and using the bfmt-utilities library to display effect data for a chosen unit.
  • The index.html file contains the markup that you see on this page. It also imports both bfmt-utilities and app.js into the browser.
