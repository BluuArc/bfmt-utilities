Basic Browser Example

Instructions

This is a basic example of how to use bfmt-utilities within the browser directly. The key thing you will want to do is import the script like so:

				<script src="https://joshuacastor.me/bfmt-utilities/dist/index.browser.min.js"></script>

(Alternatively, you could use index.browser.js if you'd like to view an unminified version.)

After adding the script, you will now have access to the bfmtUtilities variable within your JavaScript, which is the entry point for all the functions.
